# 🏙️ Billboard Organiser

**Be Seen. Be Remembered.**

Billboard Organiser (BBO) is a premium, modern landing page for a billboard advertising platform. It serves as the ultimate bridge between brands looking for high-visibility advertising space and billboard owners looking to monetize their locations.

## ✨ Key Features

-   **💎 Premium Aesthetics**: A sleek dark-themed interface with high-end glassmorphism effects and modern typography.
-   **🔄 CSS-Only Interactive UX**: A unique, JavaScript-free toggle mechanism to switch between "For Business" and "Personal" workflows.
-   **📱 Fully Responsive**: Optimized for all devices, from desktops to mobile phones.
-   **⚡ High Performance**: Lightweight structure using vanilla HTML and CSS for lightning-fast load times.
-   **📈 Informative Stats**: Clear, visual representation of platform reach and capabilities.

## 🛠️ Tech Stack

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom properties, Flexbox, Grid, and Glassmorphism.
-   **Google Fonts**: 'Outfit' for headers and 'Inter' for body text.

## 🚀 Getting Started

To view the project locally:

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
